# expression parser

Implemented using a recursive decent parser

### some examples examples
- `var a = 10;`
- `var a = sin(pi);`
- `Print(a);`
- `var c = (2*3)+((8/2)^3);`
- `var c = -34; var p = 231; Print(p / c);`

### features  
- Unary Operators `+` `-`
- Binary Operators `+` `-` `*` `/` `%` `^`
- Functions `sin` `cos` `tan` `asin` `acos` `atan` `log` `sqrt` `Int` `Print`
- Parentheses  `(` `)`  
- variables
